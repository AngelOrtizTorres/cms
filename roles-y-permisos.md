# Roles y permisos

Este proyeto contará con 3 roles principales: 

**- Administrador:** Tendrá acceso completo a todas las funcionalidades del sistema, incluyendo la gestión de usuarios, configuración del sistema y acceso a todos los datos.

**- Autor:** Tendrá acceso a funcionalidades relacionadas con la creación y edición de contenido, pero no tendrá permisos para gestionar usuarios ni configurar el sistema.

**- Usuario estándar:** Tendrá acceso limitado a ciertas funcionalidades del sistema, como la visualización de datos y la realización de tareas específicas asignadas por el administrador.

Cada rol tendrá permisos específicos que definirán qué acciones pueden realizar dentro del sistema. Estos permisos se asignarán de acuerdo con las responsabilidades y necesidades de cada rol, garantizando así una gestión eficiente y segura del sistema.

## Permisos por rol

- **Administrador:**
  - Gestionar usuarios (crear, editar, eliminar)
  - Configurar el sistema
  - Acceso completo a todos los datos
  - Realizar tareas administrativas

- **Autor:**
  - Crear y editar contenido
  - Acceso a datos relacionados con su contenido
  - No tiene permisos para gestionar usuarios ni configurar el sistema
  - Realizar tareas relacionadas con la creación de contenido

- **Usuario estándar:**
  - Visualizar datos asignados
